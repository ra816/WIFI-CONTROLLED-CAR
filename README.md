# WIFI-CONTROLLED-CAR
## 📌 Description
This project is a WiFi controlled robotic car using ESP8266 (NodeMCU). It can be controlled using a mobile device.

## 🛠 Components Used
- NodeMCU (ESP8266)
- L298 Motor Driver
- DC Motors
- Battery

## ⚙️ Working
Commands are sent from mobile via WiFi. The NodeMCU receives commands and controls the motors accordingly.

## 🎯 Applications
- Robotics learning
- Remote control vehicles

## 💡 Future Improvements
- Add camera (ESP32-CAM)
- Add obstacle avoidance
